# 💼 Business Name Generator

A simple JavaScript-based business name generator that shows a random unique name using `alert()`.


## 🚀 Features

- Click a button to generate a random business name
- Output shown via JavaScript `alert()`

## 🛠 Tech Stack

- HTML
- JavaScript

## 📁 Files

- `index.html`
- `script.js`
